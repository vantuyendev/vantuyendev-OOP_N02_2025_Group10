Welcome to Shop Management System

This is a combined web and desktop application for managing your shop.

Web Features:

  • View and manage employees, products, and customers
  • Access dashboard with real-time statistics
  • Responsive web interface

Desktop Features:

  • Full desktop application with Swing UI
  • Advanced management features
  • Local desktop integration

To use the desktop version, run the application with: --mode=desktop
